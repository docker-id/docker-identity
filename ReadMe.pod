=head1 Name

Docker Identity - Framework for Self Identifying Docker Containers

=head1 Synopsis

    > did identify dids/fortune
    {"name":"dids/fortune",...}
    > did run dids/fortune
    Fortune server running on http://0.0.0.0:12345
    > curl http://0.0.0.0:12345/
    {"fortune":"Put your Nose to the Grindstone!\n-- Amalgamated Plastic Surgeons and Toolmakers, Ltd.\n"}

=head1 Description

B<Docker Identity> is a project to make Docker containers identify themselves.
ie Declare what they do, how they do it, and how they are configured / run.

The C<did> command can run container images that meet this contract, ensuring
that they are run according to spec.

=head1 Git Repository Branches

All of the resources of the Docker Identity project are in this repository,
but on separate, unrelated branches. The important branches are:

=over

=item C<did>

The C<did> CLI command. This command is a companion to the C<docker> CLI
command itself.

=item C<dids>

A set of build command for sample indentity aware containers.

=item C<spec>

The Docker Identity specification. Describes the domain of the Identity
project, and defines the C<identity.yml> data structures that provide
identity.

=item C<web/info>

Source code for the L<https://info.dockerid.com> website.
